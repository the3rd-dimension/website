Devlog / About

The Nurse is a first-person survival horror game that focuses on environmental storytelling and resource management. This project was developed to showcase game optimization and detailed environment design


Key Technical Features:

  • State Machine based Enemy AI (Patrol, Chase, Search)
  • Detailed Environments
  • Extreamly low minimum requirements
